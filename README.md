## Description
Theta data analyse server and graphql api server.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# graphql api development
$ npm run start

# analyse server development
$ npm run analyse

# watch mode
$ npm run start:dev

# build
$ npm run build

# run graphql api server
$ node dist/main

# run analyse server
$ node dist/analyse



```
